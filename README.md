# Mirror
具有人臉辨識登入系統，以及帶有如備忘錄、天氣查詢等功能的電腦小管家

## 程式碼部分
以 Qt5 作為前端框架，並使用許多套件來實現各式各樣的功能

### 具備功能：
- 人臉辨識登入系統
- 備忘錄
- CPU、GPU 使用率顯示
- ChatGPT 問答（具有語音輸出功能）
- 即時新聞
- 天氣顯示
- 音量調整
